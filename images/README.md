# Image Assets for A & O Contracting Website

## Directory Structure

### /images
- **logo.png** - Company logo (recommended size: 200x60px, transparent background)
- **og-image.jpg** - Open Graph image for social media sharing (1200x630px)
- **why-choose-us.jpg** - Image for "Why Choose Us" section (800x600px)

### /images/hero
Place hero/banner images here:
- **demolition-hero.jpg** - Main hero background (1920x1080px or larger)
- Use high-quality images of demolition work, machinery, or completed projects
- Images should be optimized for web (compressed but high quality)

### /images/services
Service category images:
- Images representing each service type
- Recommended size: 600x400px
- Should show relevant equipment, processes, or results

### /images/portfolio
Project photos for the gallery:
- **project-1.jpg, project-2.jpg, etc.** - Thumbnail versions (600x600px)
- **project-1-full.jpg, project-2-full.jpg, etc.** - Full-size versions (1200x1200px)
- Before/after shots work well
- Include residential, commercial, and industrial projects

### /images/team
Team member photos:
- Professional headshots
- Recommended size: 400x400px
- Square format, consistent backgrounds

### /images/testimonials
Client photos:
- **client-1.jpg, client-2.jpg, etc.**
- Small headshots (200x200px)
- Can use placeholder avatars if real photos not available

### /images/icons
SVG icons for various UI elements:
- Custom icons for services
- Social media icons (if not using Font Awesome)

## Image Optimization Tips

1. **Compress images** before uploading (use tools like TinyPNG, ImageOptim)
2. **Use WebP format** where possible for better compression
3. **Provide multiple sizes** for responsive images
4. **Add alt text** for accessibility and SEO
5. **Lazy load** images below the fold

## Placeholder Images

Until you have actual photos, you can use:
- https://placeholder.com/ - Simple placeholder generator
- https://picsum.photos/ - Random stock photos
- https://unsplash.com/ - Free high-quality stock photos
